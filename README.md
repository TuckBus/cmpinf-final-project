# cmpinf0010-final-project
Evaluates the data-based "best" neighborhood in Piitsburgh using trees per square mile, burglaries and robberies per capita, and 311 complaints per capita

## Datasets Used
| Name | Filename | Description | Link |
| ---- | -------- | ----------- | ---- |
| Demographics | `crime/demographics.csv` | Contains various demographic information from the 2010 census, including the population of each neighborhood, burglaries, and robberies | [WPRDC](https://data.wprdc.org/dataset/pgh) | 
| Trees | `trees/trees.csv` | Contains information about each tree cared for by the City of Pittsburgh, including what neighborhood it is planted in | [WPRDC](https://data.wprdc.org/dataset/city-trees) | 
| 311 Complaints | `311/311.csv` | Contains information about 311 Complaints to the City of Pittsburgh, including the neighborhood from which it originated. 311 is a hotline for non-emergency complaints to the City | [WPRDC](https://data.wprdc.org/dataset/311-datas)
